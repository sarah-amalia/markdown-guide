###### What is Hugo?
Hugo is a [static site generator](https://developer.mozilla.org/en-US/docs/Glossary/SSG) (SSG) which is commonly used for technical documentation (like this page), and other similar web-driven content like landing pages, news, or project sites. Hugo renders a complete site extremely fast in milliseconds. Static site generators are an alternative to content management systems (CMS) like WordPress.

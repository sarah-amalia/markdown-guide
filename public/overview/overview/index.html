Understanding Markdown

Markdown is a text-to-HTML conversion language. It was designed for easy-to-read, write, and edit by not struggling on heavy-code. So us as writers will be focus more on the content rather than the format code. Writing in Markdown on ly requires few symbols to customize.

Markdoown was created by John Gruber in 2004.
